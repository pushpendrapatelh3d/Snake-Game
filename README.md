# 🐍 Snake Game

A simple Snake Game built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

🔗 **Play Here:** tiny-kelpie-ff48e1.netlify.app

## Features

* Score tracking
* Random food generation
* Snake growth mechanics
* Collision detection
* Game Over functionality
* Sound effects
* Responsive UI

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Clone the repository

```bash
git clone https://github.com/pushpendrapatelh3d/Snake-Game
```

2. Open `index.html` in your browser.

## Controls

* ↑ Move Up
* ↓ Move Down
* ← Move Left
* → Move Right
